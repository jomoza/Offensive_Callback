Import offline-collected dealer data into the 5ELG database.

What is this for?

If you have deployed a Dealer Proxy — an alternative dealer implementation running in a separate environment (PHP, Python, Bash, etc.) — it can generate CSV exports of the collected data. Upload that CSV here to import the captured sessions into 5ELG's database for unified analysis alongside live browser traffic.

This is especially useful for offline collection (air-gapped systems, local network traps, phishing proxies) where the dealer cannot communicate directly with this server in real time.
